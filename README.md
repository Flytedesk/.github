# Shared GitHub Workflows
Reusable GitHub Actions workflows for Flytedesk repositories.
## Included reusable workflows
- `reusable-ruby-gem-release-pr.yml`
  - Runs release-please for Ruby gems to open/update a release PR and bump version/changelog.
- `reusable-ruby-gem-publish.yml`
  - Builds and publishes a gem to RubyGems using `RUBYGEMS_API_KEY`.
